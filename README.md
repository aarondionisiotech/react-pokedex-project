# 🐱‍👤 Pokédex React



Una semplice applicazione web sviluppata con React che mostra i Pokémon starter dalla Prima alla Quarta Generazione. I dati vengono recuperati tramite PokéAPI e possono essere cercati e filtrati facilmente.



## ✨ Funzionalità



- Ricerca dei Pokémon per nome

- Filtro per tipo (Erba, Fuoco o Acqua)

- Modalità Artwork ufficiale e Sprite Pixel

- Visualizzazione della versione Shiny

- Badge della generazione di appartenenza

- Design responsive per desktop, tablet e smartphone

- Dati caricati in tempo reale tramite PokéAPI



---



## 🖼️ Come funziona



L'app mostra una raccolta di carte Pokémon contenenti:



- Immagine del Pokémon

- Nome

- Numero Pokédex

- Tipo principale

- Generazione

- Pulsante per visualizzare la versione Shiny



Nella parte superiore della pagina è presente una barra di ricerca e un menu per filtrare i Pokémon.

## 🚀 Installazione



Clona il repository:



```bash

git clone https://github.com/tuo-username/pokedex-react.git

```



Entra nella cartella del progetto:



```bash

cd pokedex-react

```



Installa le dipendenze:



```bash

npm install

```



Avvia il server di sviluppo:



```bash

npm run dev

```
